# Сайт-открытка для жены 💖

Анимированная веб-открытка для **Альбины**.

## Что внутри
- Переливающийся градиентный фон
- Имя «Альбина» большими буквами с анимацией появления
- Летящие вверх сердечки
- Кнопка «Нажми меня ✨» — фейерверк сердечек + случайное милое сообщение

## Где открыть
🔗 https://fosttt.github.io/for-wife/

## Как изменить
- **Имя** — в `index.html` строка `<h1 id="name">Альбина</h1>`
- **Сообщения кнопки** — в `index.html` массив `messages = [...]` внизу в `<script>`

## Хостинг
Опубликовано через **GitHub Pages** (ветка `main`, корень репозитория).
Репозиторий публичный — это обязательное условие бесплатного GitHub Pages.

## Деплой обновлений
```bash
git commit -am "что изменил"
git push
```
Сайт пересобирается автоматически за ~30–60 секунд.
